# DiplodocusInterpreter
### Main code interpreter for *Diplodocus* language

## Usage
### Command
```
dii -i <FileName>
```
- ```-i``` input file name
- Optional arguments:
    - ```-h``` show help message

### Example
```
dii -i prog.di
```
